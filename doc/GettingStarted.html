Курс "MATLAB для преподавателей"

Описание

Учебный курс для обучения преподавателей высших учебных заведений основам MATLAB и преподаванию с помощью MATLAB. Включает в себя следующие разделы:
  • Введение в MATLAB
  • Анализ данных в MATLAB
  • Программирование и математика
  • Создание лекций
  • Проверка домашних заданий
  • Лабораторные работы
  • материалы для наполнения курсов
Курс содержит файлы для занятий в классе и самостоятельной работы.
Содержание курса и часы

Системные требования

MATLAB версии не ниже R2018b или MATLAB Online*. Также требуется доступ к MATLAB Drive и MATLAB Grader.
Необходимые пакеты:
  • Curve Fitting Toolbox
  • Optimization Toolbox
  • Signal Processing Toolbox
  • Statistics and Machine Learning Toolbox
  • Symbolic Math Toolbox
* разделы "2.3. Лабораторные работы", "1.2.15. Восстановление пропусков и прогнозирование", "1.2.18. Анализ и обработка аудиофайла в MATLAB" рекомендуется проходить в настольной версии MATLAB, потому что на данный момент только она содержит необходимые приложения (Apps)

Начало работы

Чтобы начать работу, запустите приложение Курс "MATLAB для преподавателей" на вкладке APPS главного окна MATLAB.
Или выполните команду
edu_course_app
Откроется окно приложения:
Нажмите "Установить курс в текущую папку" чтобы начать. При этом в текущую рабочую папку будут скопированы рабочие файлы курса (папка EduCourseFiles), все подкаталоги автоматически добавятся в путь (path) MATLAB и откроется содержание курса.
Чтобы продолжить работу над курсом после перезапуска MATLAB, сделайте рабочей папкой директорию, в которой хранится папка EduCourseFiles и снова запустите приложение.
После этого нажмите на кнопку "Приступить", после чего все подкаталоги автоматически добавятся в путь (path) MATLAB и откроется содержание курса.
После завершения курса рекомендуется запустить приложение и нажать "Заархивировать файлы", что поместит все рабочие файлы с выполненным заданиями в zip-архив, который вы можете забрать из учебного класса, чтобы использовать свои наработки в дальнейшем.

Получение помощи

Нажмите на кнопку в правом верхнем углу программы, чтобы открыть документацию к курсу.
